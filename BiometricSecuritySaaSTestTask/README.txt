вход:
файл photos.txt
текстовой файл, содержит несколько строчек следующего формата

PHOTO1.JPG	name1	name2	1	name3	2
PHOTO2.JPG	2	name1
....
поля разделены знаком табуляции. первое поле - название фотографии, потом идут имена людей или порядковые номера

файл names.txt
1.	name11
2.	name12+
...

после числа стоит точка и знак табуляции. содержит информацию о том, какой порядковый номер соответствует какому имени.

результат:
photos-final.txt
содержит те же строчки, что и входной файл, но порядковые номера заменить на имена, соответственно файлу names.txt

строчки не соответствующие правильному формату должны быть игнорированы

пример:
для данных указанных выше должно получиться 

PHOTO1.JPG name1 name2 name11 name3 name12
PHOTO2.JPG 2 name1
