Создать веб-приложение позволяющее вносить и хранить в базе технические статьи. Каждая статья имеет автора и относящиеся к ней тэги. Использовать NHibenate для доступа к базе. tinyMCE для ввода статей. Для создания пользователй использоватей Membership.
Сайт должен поддерживать поиск по тэгам (поле ввода с Ajax auto completion). Поддерживать две темы и два языка. Использовать Master Page.
