Functionality

Contains
  • MatrixExtensions.Conversion.* - ToOtherArray, ToJagged, ToNumeric (Int, Double, ...)
  • MatrixExtensions.Manipulation.* - Append, Stack, ToColumnWiseArray, ToRowwiseArray
  • MatrixExtensions.Operations.* - Multiply, Substract, Add, MaxPosition, PointwiseMultiply, DivideEachElement, SumPerColumn, SumPerRow
  • MatrixExtensions.Equality.* - EqualsElements
  • MatrixExtensions.Generic.* - IsEmpty, RowCount, ColumnCount, GetRow, GetColumn,GetLastColumn, GetRowEnumeration, GetColumnEnumeration, ToListOfLists
See source code and test suite.

Behavior

See unit tests.
double[4] append int[5] = double[9]
int[4] append long[4,5] = long[4,6]
int[4] stack double[2,4] = double[3,4]
double[2,2] add long[2,2] = double[2,2]

int[3] stack double[2,4] = exception
decimal[4] append long[2,3] = exception

decimal[3,3] substract  float[3,3] = compilation error
